# LINE Bot Dispatcher 完整功能列表

## 🎵 娛樂功能

### 影片推薦
- **讚嘆姿儀嬤** → 推薦 ATEEZ 熱門 YouTube MV
- **讚嘆宜庭嬤** → 推薦 One Ok Rock 影片
- **讚嘆神豬** → 推薦 Twice MV

### 圖片發送（自動補圖機制）
- **姿儀嬤顯靈** → ATEEZ 圖片
- **宜庭嬤顯靈** → One Ok Rock 圖片
- **神豬顯靈** → Twice TZUYU 圖片
- **早安** → 早安問候圖片
- **晚安** → 晚安問候圖片
- **生日快樂** → 海綿寶寶生日梗圖/影片

## 📈 股票功能

- **分析XX** → 台股/美股五維分析（基本面、技術面、風險、趨勢、建議）
- **#推薦** / **#選股** → 每日股市推薦
- **#股票代號** → 即時報價查詢（yfinance + Gemini 補充）

## 🍽️ 生活功能

- **不知道吃什麼** / **不知道吃啥** → 隨機推薦菜系
- **地點 + 吃什麼** → 推薦該地區餐廳（例如：台中大甲 吃什麼）
- **地點 + 天氣** → 查詢天氣 + 未來三天降雨機率

## 🔮 命理功能（!指令）

- **!今日** / **!好日子** / **!today** → 農民曆宜忌
- **!星座 YYYY-MM-DD** → 查詢星座
- **!運勢 星座名稱** → 今日星座運勢
- **!本週運勢 星座名稱** → 本週星座運勢
- **!塔羅 問題** → 塔羅占卜
- **!抽籤** → 月老籤詩解析
- **!擲筊** / **!擲茭** → 擲筊
- **!讚美我** / **!誇獎我** / **!稱讚我** / **!拍馬屁** → AI 彩虹屁

## 📅 Google Workspace（-指令）

### 行事曆
- **-日曆** → 顯示今日行程
- **-週曆** → 顯示本週行程
- **-月曆** → 顯示本月行程
- **-新增 日期 時間 事件** → 新增行程

### 待辦/購物清單
- **-待辦** → 顯示所有待辦事項
- **-新待辦 內容** → 新增待辦
- **-完成 編號** → 標記完成
- **-清單** → 同「待辦」

### 記事本
- **>doc** → 記事本功能說明
- **>doc讀 編號** → 讀取記事
- **>doc寫 編號 內容** → 寫入記事
- **>doc列表** → 列出所有記事本

## 🤖 AI 對話

- **elior + 問題** → Gemini AI 對話（自動 Google Search 補充）
- **%關鍵字** → 一般資訊查詢（Gemini + Google Search）

## 📱 特殊功能

- **UberEats 訂單連結** → 自動追蹤訂單狀態（每 60 秒更新）
- **>?** / **>？** → 顯示所有指令總覽
- **>#** / **>%** / **>-** / **>!** / **>doc** → 各類功能詳細說明

## 🎯 自動化機制

- **圖庫自動補充**：6 個圖庫低於 30 張時自動背景補充 20 張
- **股票分析**：整合 yfinance + Gemini 2.5-flash
- **命理計算**：整合農曆轉換、星座查詢、八字排盤
- **Google Workspace**：透過 Maton API Gateway 操作
