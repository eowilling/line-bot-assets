# LINE Bot Dispatcher - 所有觸發關鍵字與指令

## 🎵 娛樂相關

### 影片推薦
- `讚嘆姿儀嬤` → ATEEZ 熱門 YouTube MV
- `讚嘆宜庭嬤` → One Ok Rock 影片
- `讚嘆神豬` → Twice MV

### 圖片發送
- `姿儀嬤顯靈` → ATEEZ 圖片
- `宜庭嬤顯靈` → One Ok Rock 圖片
- `神豬顯靈` → Twice TZUYU 圖片
- `早安` → 早安圖片
- `晚安` → 晚安圖片
- `生日快樂` → 海綿寶寶生日梗圖（包含 @user 發圖片，無 @ 發影片）
- `happybirthday` → 同上

## 📈 股票相關

### 分析與推薦
- `分析 股票代號` → 五維分析（基本面、技術面、風險、趨勢、建議）
- `#推薦` → 每日股市推薦
- `#選股` → 同上

### 即時報價
- `#股票代號` → 即時報價查詢（例如：#2330、#AAPL）

## 🍽️ 生活相關

### 餐廳推薦
- `不知道吃什麼` → 隨機推薦菜系
- `不知道吃啥` → 同上
- `地點 吃什麼` → 推薦該地區餐廳（例如：台中大甲 吃什麼）
- `地點 吃啥` → 同上

### 天氣查詢
- `地點 天氣` → 天氣 + 未來三天降雨機率（例如：台中苑裡 天氣）

## 🔮 命理相關（!前綴）

### 農民曆與運勢
- `!今日` → 農民曆宜忌
- `!好日子` → 同上
- `!today` → 同上

### 星座
- `!星座 YYYY-MM-DD` → 查詢星座
- `!運勢 星座名稱` → 今日星座運勢
- `!本週運勢 星座名稱` → 本週星座運勢

### 占卜
- `!塔羅 問題` → 塔羅占卜
- `!抽籤` → 月老籤詩解析
- `!擲筊` → 擲筊
- `!擲茭` → 同上

### 彩虹屁
- `!讚美我` → AI 彩虹屁
- `!誇獎我` → 同上
- `!稱讚我` → 同上
- `!拍馬屁` → 同上

## 📅 Google 行事曆（-前綴）

### 查詢
- `-日曆` → 今日行程
- `-週曆` → 本週行程
- `-月曆` → 本月行程

### 新增
- `-新增 日期 時間 事件` → 新增行程

### 待辦清單
- `-待辦` → 顯示所有待辦
- `-新待辦 內容` → 新增待辦
- `-完成 編號` → 標記完成
- `-清單` → 同「待辦」

## 📝 Google Docs 記事本

### 記事本操作
- `>doc` → 功能說明
- `>doc讀 編號` → 讀取記事
- `>doc寫 編號 內容` → 寫入記事
- `>doc列表` → 列出所有記事本
- `-doc undo` → 撤銷上次寫入

## 🤖 AI 對話

- `elior 問題` → Gemini AI 對話（自動 Google Search）
- `%關鍵字` → 一般資訊查詢（Gemini + Google Search）

## 📱 特殊功能

### 說明文件
- `>?` → 所有指令總覽
- `>？` → 同上（全形）
- `>#` → 股票功能說明
- `>＃` → 同上（全形）
- `>%` → 資訊查詢說明
- `>％` → 同上（全形）
- `>-` → Google Workspace 說明
- `>－` → 同上（全形）
- `>!` → 命理功能說明
- `>！` → 同上（全形）

### UberEats 追蹤
- `包含 ubereats 和 orders 的網址` → 自動追蹤訂單（每 60 秒更新）

## 🔤 符號說明

- `#` → 股票查詢前綴（支援全形 ＃）
- `%` → 資訊查詢前綴（支援全形 ％）
- `-` → Google Workspace 前綴（支援全形 －）
- `!` → 命理功能前綴（支援全形 ！）
- `>` → 說明文件前綴
- `elior` → AI 對話觸發詞

## 📋 特殊規則

1. **股票代號格式**：台股純數字（2330）、美股字母（AAPL）
2. **日期格式**：YYYY-MM-DD 或 YYYY/MM/DD
3. **時間格式**：HH:MM（24小時制）
4. **地點要求**：至少 2 個字（例如：台中、大甲）
5. **生日快樂**：精確匹配優先，包含 @ 發圖片，否則發影片
6. **全形半形**：# % - ! 都支援全形/半形
